# 分诊标签

工程技能使用五种标准分诊角色。下表定义这些角色在本仓库议题跟踪器中的实际标签。

| 标准角色 | 仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可由 Agent 独立处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不予处理 |

技能提到某个分诊角色时，使用本表对应的仓库标签。
