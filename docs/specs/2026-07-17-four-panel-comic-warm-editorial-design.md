# 四格漫画原型：柔雾编辑部视觉优化

## 目标

将 `prototypes/four-panel-comic/index.html` 从高饱和、厚描边的游戏化风格，调整为轻松、温馨且以漫画内容为主角的阅读体验。功能、页面结构、文案、数据与交互行为保持不变。

## 已确认的视觉方向

采用「柔雾编辑部」：以暖象牙白作为主要纸张底色，低饱和鼠尾草绿作为主强调色，搭配深苔绿文字与柔和灰绿分隔线。卡片、导航和阅读器使用轻薄边框、克制圆角与低扩散阴影，去除立体厚底与高饱和奖励感。

## 背景素材

使用本次生成视觉板中 B 方向的柔和纸张与植物氛围作为原型外层背景参考。实现时将该图裁切并弱化为手机壳外的背景层；不覆盖手机模拟器内的内容，不干扰漫画、文字和交互控件的可读性。

## 视觉改动范围

- 替换页面色彩 token、字体组合、边框、阴影、圆角与动效节奏。
- 将首页、发现页、我的页面、阅读器及弹窗的高饱和色统一为柔雾编辑部配色。
- 将当前 emoji 图标保留为既有原型信息符号，不更改事件、可点击区域或状态切换逻辑。
- 为外层页面添加低对比背景图层；手机容器继续保持内容隔离。
- 保留现有布局尺寸、Tab、轮播、收藏、点赞、阅读器滑动和弹窗交互。

## 非目标

- 不新增功能、页面、数据、接口或第三方依赖。
- 不修改 JavaScript 交互逻辑或现有文本内容。
- 不将生成图中的人物、漫画内容或标记作为原型 UI 内容。

## 验收标准

- 原型呈现低饱和、温馨、安静的漫画阅读氛围，不再呈现多邻国式游戏化视觉。
- 任何既有页面与交互均可继续使用，且布局不发生功能性变化。
- 外层背景不降低手机容器内文字、按钮、漫画格或状态信息的可读性。
- 窄屏下页面仍可完整显示，动效尊重减少动态效果偏好。
